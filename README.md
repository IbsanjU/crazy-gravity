## Getting Started

1.  Clone the repo

2.  Install dependencies:

        yarn

    or

        npm install

3.  Run webpack:

        npm start

Your canvas piece should open up automatically at http://localhost:3000 and you should see 'HTML CANVAS BOILERPLATE' on hover.
